Minhas Habilidades

CSS

CSS é a linguagem usada para descrever a apresentação de um documento HTML. Ela define como os elementos HTML devem ser exibidos em termos de layout, cores, fontes e outros aspectos visuais.

HTML

HTML é a linguagem de marcação usada para criar a estrutura de uma página web. Ela define os elementos que compõem a página, como títulos, parágrafos, links, imagens e outros tipos de conteúdo.

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade e funcionalidades dinâmicas às páginas web.
